# ConfigManager class to provide config with json file

## Build

```shell
make
```

## Install

```shell
sudo make install
```
## Uninstall

```shell
sudo make uninstall
```

## Files to install
| Filename | Location |
| -------- | -------- |
| libconfigmanagercpp.so | /usr/lib/libconfigmanagercpp.so |
| ConfigManager.hpp | /usr/include/ConfigManager.hpp |
| json.hpp | /usr/include/json.hpp |
